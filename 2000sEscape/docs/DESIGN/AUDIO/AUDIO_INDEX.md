# 逃离千禧年 - 音乐需求总览

## 项目音乐定位

《逃离千禧年》是一款怪核/池核风格的2D像素搜打撤游戏，音乐需要传达：
- **怀旧感**: 2000年代中国童年的记忆碎片
- **孤寂感**: 被困在梦境中的孤独与迷失
- **幻梦感**: 记忆模糊、现实扭曲的超现实氛围
- **微妙的不安**: 熟悉中透着陌生，温暖中带着诡异

## 音乐风格指南

### 核心风格关键词
- **Lo-fi / 低保真**: 模拟磁带、VHS的低质感
- **Ambient / 氛围音乐**: 空间感、沉浸感
- **Synthwave / 合成器**: 千禧年的电子感
- **Drone / 长音**: 梦境的延展与停滞
- **Environmental / 环境音**: 场景的沉浸

### 配器建议
- 老式合成器 (FM合成、模拟合成器音色)
- 音乐盒 / 八音盒音色
- 磁带失真 / 噪音质感
- 柔和的钢琴 / 电钢琴
- 环境采样 (雨声、电流声、远处的电视声)
- 中国传统乐器元素 (二胡、古筝的Lo-fi处理)

### 情绪光谱

```
安静/忧伤 ←——————————→ 紧张/不安
    │
    │  [梦境大厅]     [战斗警报]
    │      ↓              ↓
    │   孤寂梦幻      紧凑节奏
    │
    │  [探索阶段]     [撤离阶段]
    │      ↓              ↓
```

## 音乐分层设计

部分BGM需要支持动态分层，根据游戏状态实时切换：

### 分层结构示例（关卡内）
```
Layer A: 基础氛围层 (始终播放)
Layer B: 探索旋律层 (安全时播放)
Layer C: 紧张节奏层 (敌人接近时渐入)
Layer D: 战斗层 (战斗中播放)
```

### 过渡方式
- **硬切**: 用于紧急战斗开始
- **淡入淡出**: 2-4秒渐变
- **垂直混音**: 实时调整各层音量

## 技术规格

### 文件格式
- 主格式: OGG Vorbis (循环优化)
- 备份格式: M4A (兼容性)
- 采样率: 44.1kHz
- 比特率: 128-192 kbps

### 循环点标记
- 每首BGM需标注循环起始点和结束点
- 前奏 (Intro): 4-8小节
- 循环主体 (Loop Body): 16-32小节
- 尾声 (Outro): 可选，用于非循环结束

### 音量基准
- BGM基准: -12dB
- 峰值不超过: -6dB
- 需预留3-4dB动态余量给音效

## SUNO生成工作流

1. 使用各文档中的Prompt生成初版
2. 筛选2-3个候选版本
3. 根据以下标准评估:
   - 风格匹配度
   - 循环可行性
   - 情绪传达准确度
4. 必要时使用Extend功能延长
5. 后期处理: 循环点编辑、音量标准化
