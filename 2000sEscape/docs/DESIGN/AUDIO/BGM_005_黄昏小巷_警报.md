# BGM-005: 黄昏小巷 - 警报

## 基本信息

| 属性 | 值 |
|------|-----|
| 编号 | BGM-005 |
| 名称 | 黄昏小巷 - 警报 |
| 用途 | 童年小巷关卡 - 战斗/被追逐阶段 |
| 目标时长 | 90-120秒 |
| 循环 | 是 |
| 优先级 | P0 (核心) |

## 使用场景

**童年小巷 - 警报/战斗阶段**
- 敌人发现玩家，进入追逐或战斗状态
- 玩家需要战斗或躲避
- 紧张度最高

**场景描述**:
> 它们发现你了。那些模糊的黑影从角落里涌现，带着扭曲的轮廓向你冲来。心跳加速，你必须战斗或逃跑。熟悉的街道此刻变得陌生，每一个转角都可能是陷阱，也可能是生路。

## 风格参考

### 参考风格
- **Y2K Disco**: 千禧年迪斯科 - 敌人的派对时间
- **Creepy Happy**: 诡异欢快 - 欢快的音乐配上危险的处境
- **Glitch Disco**: 故障迪斯科 - 带着损坏感的舞曲

### 核心概念
> 这是**敌人**的BGM。敌人追着你打，敌人很开心。对它们来说，这是派对。

### 参考作品
- *Yume Nikki* 的追杀者音乐 (欢快但诡异)
- *Earthbound* 的战斗音乐 (那种奇怪的欢快感)
- *Cruelty Squad* 的诡异舞曲
- 70-80年代迪斯科 (但更怪异)
- 千禧年复兴迪斯科

### 配器设想
```
核心:
- 迪斯科节拍 (four-on-the-floor)
- 欢快的贝斯线 (但有点怪)
- 弦乐/铜管的 stabs (像70年代迪斯科)

氛围:
- 欢快、舞动感
- 但带着某种不对劲
- 像是"敌人开派对"的感觉

诡异元素:
- 偶尔的故障/glitch
- 音符偶尔走调
- 节奏偶尔卡顿
- 隐约的失真
- 欢快中带着危险感

千禧年元素:
- 早期数字合成器的音色
- 磁带质感
- 像是2000年初舞厅的音乐
```

```

```

## 循环设计

```
[紧张前奏 4s] → [A段 16s] → [B段 16s] → [A'段 16s] → [C段 16s] → 循环
```

**设计要点**:
- 需要持续的高紧张感
- 不能有"放松"的时刻
- 节奏需要驱动感但不至于太快

## SUNO Prompt

### 主Prompt (推荐 - 敌人的迪斯科派对)

```
Style: Y2K disco, creepy happy, enemy party music, unsettling dance, 2000s dance floor horror
Instruments: Disco beat (four-on-the-floor), funky bassline, string/horn stabs, vintage synth, occasional glitches
Mood: Happy hunting, party time for monsters, fun but wrong, dance while you chase, uncanny joy
Tempo: 120-130 BPM, danceable
Structure: Looping dance track with slight wrongness
Production: Early 2000s digital warmth, slight lo-fi, tape texture, occasional digital artifacts
Description: DISCO music for when enemies chase the player. But here's the twist - the ENEMIES are having fun. This is THEIR party. You're running for your life while they dance. Upbeat 70s-80s style disco with string stabs and funky bass, but something feels slightly off. Like a carnival where the clowns are hunting you. The music is happy - that's what makes it terrifying. Not horror music - PARTY music. You're just not invited.
```

### 备选Prompt 1 (更诡异迪斯科)

```
Style: Creepy disco, uncanny dance floor, wrong happiness, distorted funk
Instruments: Disco drums, slightly detuned synth bass, off-key string hits, weird synth melodies
Mood: Unsettling joy, something's wrong at the party, smiling predators
Tempo: 125 BPM
Production: Lo-fi but danceable, occasional pitch warble, slight distortion on happy sounds
Description: Upbeat dance music that feels wrong. The bass is funky but a bit off. The strings hit but they're slightly out of tune. It sounds like a party - that's the horror. The enemies are dancing while they hunt you. 2000s nostalgia meets 70s disco meets something deeply unsettling. Happy music for a dangerous situation.
```

### 备选Prompt 2 (更千禧年舞厅感)

```
Style: Y2K club music, early 2000s dance floor, nostalgic danger disco
Instruments: Electronic disco beat, synth bass, filtered disco strings, retro synth leads
Mood: Neon danger, sweaty dance floor panic, nostalgia nightmare party
Tempo: 128 BPM, club tempo
Production: Early 2000s club sound, compressed but warm, radio-ready but weird
Description: Like walking into a dance club in 2002 but everyone there wants to hurt you. The music is genuinely good - that's the problem. You want to dance but you need to run. The enemies are vibing to this. Four-on-the-floor beat, disco strings, funky bass. The horror is in how much FUN it sounds like.
```

### 备选Prompt 3 (故障迪斯科 - 原故障蒸汽波方案保留)

```
Style: Glitch vaporwave, broken lo-fi, Y2K horror, corrupted nostalgia, PS1 horror game music
Instruments: Distorted synth pads, stuttering broken beats, VHS noise texture, tape wow and flutter, low-bitrate digital artifacts, mangled childhood melodies
Mood: Corrupted danger, digital decay, urgent malfunction, familiar but broken, panic through static
Tempo: 110-130 BPM with stuttering/glitching feel
Structure: Tension loop with intentional "errors", skipping beats, sudden dropouts
Production: Heavy lo-fi, tape saturation, sample rate reduction, datamoshing effect, bitcrushing, VHS tracking errors
Description: Combat chase music with a broken, corrupted feel. Like a damaged VHS tape of a childhood memory suddenly turning into a nightmare. Stuttering broken beats, distorted synths that sound like they're from an old game console, sudden dropouts and glitches. The feeling of something wrong in familiar territory. Not clean cyberpunk - dirty, broken, year 2000 digital decay.
```

## 与游戏节奏的配合

```
游戏状态                音乐表现
────────────────────────────────────
敌人发现玩家      →     立即切换，前奏紧张
战斗进行中        →     持续高能
玩家躲避成功      →     渐变回探索
多个敌人同时      →     可以加强节奏层
Boss战 (如有)     →     可能需要变奏
```

## 验收标准

### 必须满足
- [ ] 时长 ≥ 90秒，可循环
- [ ] 明确的紧张/危险感
- [ ] 有驱动感的节奏
- [ ] 与BGM-004可衔接
- [ ] 不会让玩家疲劳

### 优先满足
- [ ] 保持游戏的怪核氛围
- [ ] 有"千禧年迪斯科"的舞动感 (敌人派对感)
- [ ] 欢快但诡异 (敌人很开心，玩家很危险)
- [ ] 不是恐怖音乐，而是"错误的欢快音乐"

### 加分项
- [ ] 有独特的"警报"特征音
- [ ] 适合长时间战斗不重复
- [ ] 有恐惧但不绝望的感觉
