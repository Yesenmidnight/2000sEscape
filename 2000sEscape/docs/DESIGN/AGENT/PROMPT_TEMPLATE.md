# 对话记录提示词模板

> 在任何AI对话中使用以下提示词来启用对话记录功能

---

## 🎯 完整版提示词（推荐首次使用）

```
我需要你帮我记录我们的对话。

请按照以下格式，把我们的对话保存到本地文件：

# 对话记录 #{序号}: {主题}

**时间**: {当前日期时间}
**类型**: {任务类型}
**状态**: ✅/⏳/❌

---

## 用户请求

> {用户发送的完整文本}

---

## AI 响应

{AI的完整响应内容，包括分析过程、执行步骤、代码、结果}

---

## 备注

{其他重要信息}

保存位置：{你的路径，例如：docs/AGENT/}
文件命名：{日期}_{序号}_{主题}.md

当我说"保存对话"时，请创建新的记录文件。
```

---

## ⚡ 精简版提示词（已建立关系后）

```
保存对话到 {路径}，格式如下：
- 记录我的完整请求（引用格式）
- 记录你的完整响应
- 文件名：{日期}_{序号}_{主题}.md
```

---

## 📋 针对本项目的提示词

```
请帮我记录对话到 docs/DESIGN/AGENT/ 文件夹。

格式要求：
1. 记录我的每个完整请求（用 > 引用格式）
2. 记录你的完整响应（分析、代码、结果）
3. 多轮对话都要记录
4. 文件名格式：2024-02-08_{序号}_{主题}.md

当我说"保存对话"时创建新文件。
```

---

## 💡 使用技巧

### 触发保存的关键词
- "保存对话"
- "记录这次对话"
- "保存到AGENT文件夹"

### 指定主题
- "保存对话：插件开发"
- "保存对话：bug修复"

### 追加到现有文件
- "追加到上一个对话记录"

---

## 🔄 跨会话使用

如果你在一个新的AI会话中想要继续记录：

```
我之前在另一个对话中记录了一些内容。
请继续帮我记录对话，保存到同样的位置：
{你的路径}

文件序号从 {N} 开始。
```

---

## 示例：本项目实际使用的提示词

### 首次请求
> 我想要保存我们的每个对话到本地，我们可以自动进行这个吗？我想把我们的每个对话保存到DESIGN文件夹下的AGENT文件夹下。

### 格式细化
> 我想在每次我们记录的时候把我的具体请求（我发送的对话框里的所有文本）和你的反馈结果（可以省掉thinking的地方但是也应当大部分记录）都记录下来。

---

## ⚠️ 注意事项

1. **AI无法自动触发** - 必须由你主动说"保存对话"
2. **每次会话需要重新说明** - 新对话需要重新发送提示词
3. **路径需要存在** - 确保目标文件夹已创建，或让AI创建
4. **文件序号** - 可以让AI自动递增，或手动指定
