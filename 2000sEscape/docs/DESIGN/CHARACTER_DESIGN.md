# 角色设计策划案 - 逃离千禧年 (2000sEscape)

**版本**: 4.0
**日期**: 2026-02-08
**用途**: 角色/NPC/敌人的叙事与玩法设计
**核心方法论**: 恐惧概念提取 → 视觉符号化 → 夸张放大 → 极简其他

---

## 概述

### 核心设计哲学

**小孩子只记得给他最大恐惧的那个概念，然后把那个概念凝聚成实体。**

这个实体被孩子们称为"恶魔"。

**设计方法论**:

```
原型对象 → 提取恐惧核心 → 视觉转化为符号 → 夸张放大 → 极简其他元素
   ↓            ↓              ↓           ↓          ↓
  老师    →   看+打    →   眼睛+戒尺  →  巨大化   →  黑色剪影
```

**例子**：
- 千禧年小孩子怕老师 → 老师可以体罚 → 核心恐惧是"看"(被发现)和"打"(戒尺)
- "看" → 巨大的眼睛代替头颅
- "打" → 戒尺夸张成大砍刀
- 其他 → 黑色西装剪影，不需要细节

### 设计原则

1. **提取核心** - 找出恐惧对象最可怕的1-2个概念
2. **符号化** - 把概念转化为视觉符号（眼睛、手、嘴等）
3. **夸张放大** - 符号部位不成比例地巨大
4. **极简其他** - 非核心部位用最简单的剪影/颜色处理
5. **没有"正常人形"** - 永远是符号的组合，不是完整的人

---

# 一、主角设计

## 1.1 主角 - "你"

### 概念
主角是**童年的你**——正在重新经历那些被遗忘的童年记忆。你是记忆碎片中的存在，所以外形也是模糊的。

### 视觉表现（应用设计方法论）

| 部位 | 概念 | 符号化 | 处理 |
|------|------|--------|------|
| 整体 | 记忆中的自己 | 孩童剪影 | 模糊边缘 |
| 头部 | "看不清的脸" | 无面 | 空白或朦胧 |
| 身体 | "小小的我" | 矮小身形 | 比例偏小 |
| 边缘 | "正在回忆" | 朦胧效果 | 轻微发光/模糊 |

**最终形象**: 矮小的无面孩童剪影，边缘有"正在回忆"的朦胧效果

### 核心属性

| 属性 | 描述 | 初始值 |
|------|------|--------|
| **勇气** | 相当于HP，太低会"吓跑" | 100 |
| **记忆碎片** | 收集品数量 | 0 |
| **躲藏次数** | 躲进柜子/桌子下的次数 | 3/关 |

### 核心能力

| 能力 | 描述 | 玩法 |
|------|------|------|
| **躲藏** | 小孩子本能——躲起来 | 躲进柜子/桌子/阴影 |
| **逃跑** | 小孩子本能——跑掉 | 快速逃离危险 |
| **回头看** | 忍不住回头看 | 揭示某些东西 |
| **哭** | 最后的武器 | 暂时驱散某些存在 |

# 二、"存在"设计（替代NPC概念）

## 2.1 存在的本质

**存在 = 童年记忆中的温暖存在，以孩子眼中的符号形式出现。**

存在不是敌人，是**记忆中的谁**——但以抽象的符号呈现。

### 存在类型

| 类型 | 恐惧/温暖核心 | 符号化表现 |
|------|---------------|------------|
| **守护者** | 保护、温暖 | 柔和的符号（摇椅、光、声音） |
| **引导者** | 带路、等待 | 方向性的符号（背影、脚印） |
| **交易者** | 交换、帮助 | 交易相关的符号（柜台、手） |

---

## 2.2 童年小巷 - 存在设计

### 存在-001 摇椅奶奶

**原型**: 邻居奶奶/长辈
**类型**: 守护者

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 摇扇子、说话、等待 |
| **符号化** | 蒲扇、摇椅、声音 |
| **夸张** | 摇椅自己摇、蒲扇自己动 |
| **极简** | **看不到人形**，只有这些符号 |

**最终形象**:
- 一把空摇椅在微微摇晃
- 一把蒲扇在空中轻轻摇动（没人拿着）
- 断断续续的声音："...回来啦...吃饭了吗..."
- **没有身体/人形**

**触发感受**:
- 邻里的温暖
- 被照顾的安心
- "她还在那里"

**玩法功能**:
- 存档点（靠近后感到安心）
- 恢复勇气
- 触发内心独白

---

### 存在-002 柜台后的手

**原型**: 小卖部店主
**类型**: 交易者

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 给东西、收钱、认识你 |
| **符号化** | 手、货物、声音 |
| **夸张** | 手从阴影中伸出 |
| **极简** | 柜台后只有黑暗，看不清人形 |

**最终形象**:
- 玻璃柜台后面是一片**黑暗**
- 交易时，一只**手**从黑暗中伸出
- 货物被"推"过来，零钱被"递"过来
- 模糊的声音："...要什么...老样子..."

**触发感受**:
- 童年买零食的期待
- 熟悉的交易
- "他认识我"

**玩法功能**:
- 交易点
- 购买/兑换物品
- 环境叙事

---

### 存在-003 玩耍的影子们

**原型**: 童年玩伴
**类型**: 引导者

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 玩耍、跑动、笑声 |
| **符号化** | 影子、笑声、弹珠/皮筋 |
| **夸张** | 影子脱离实体独立存在 |
| **极简** | 只有影子，没有身体 |

**最终形象**:
- 地面上有几个**孩子的影子**在玩耍
- 影子在跳皮筋、弹弹珠——但**看不到产生影子的身体**
- 能听到孩子的笑声
- 你靠近，影子就跑远

**触发感受**:
- 和小伙伴玩耍的快乐
- "我也曾经这样"
- 但追不上

**玩法功能**:
- 环境氛围
- 靠近触发内心独白
- 引导探索方向

---

### 存在-004 那个背影

**原型**: 重要的某人（核心谜团）
**类型**: 引导者

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 在前面、很重要、追不上 |
| **符号化** | 背影、远去、消失 |
| **夸张** | 永远保持距离 |
| **极简** | **只有背影，看不到正面** |

**最终形象**:
- 一个模糊的**背影**
- 站在远处、走廊尽头、窗边
- 你靠近，它就走远或消失
- **永远看不到正面/脸**

**触发感受**:
- 这是谁？
- 很重要的人
- 核心谜团

**玩法功能**:
- 剧情关键点
- 追逐/探索的目标
- 多次出现逐渐揭示

# 三、"恐惧"设计（替代敌人概念）

## 3.1 恐惧的本质

**恐惧 = 童年真实的恐惧对象，以夸张的符号形式出现。**

小孩子的恐惧是**概念化的**——他们怕的不是"那个人"，而是那个人代表的"恐惧概念"。

### 恐惧类型

| 类型 | 核心恐惧 | 符号化表现 |
|------|----------|------------|
| **追赶** | 被抓住、被惩罚 | 追逐的符号（手、影子） |
| **注视** | 被发现、被监视 | 眼睛、视线 |
| **包围** | 被孤立、被围观 | 多个符号围拢 |
| **蔓延** | 怕黑、怕未知 | 黑暗扩散 |

---

## 3.2 童年小巷 - 恐惧设计

### 恐惧-001 追赶的手

**原型恐惧**: 怕被抓住、怕被骂
**类型**: 追赶

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 抓住、拖回去、惩罚 |
| **符号化** | 巨大的手 |
| **夸张** | 手比身体还大，从黑暗中伸出 |
| **极简** | 只有一只手，其他都是黑暗 |

**最终形象**:
- 身后黑暗中伸出一只**巨大的手**
- 手指张开，在追赶
- 速度不快，但一直在追
- **没有身体，只有手**

**触发感受**:
- 做错事怕被发现
- 怕被抓住
- "有什么在追我"

**应对**:
- 跑
- 躲进柜子/桌子
- 躲到它消失

---

### 恐惧-002 楼道的眼睛

**原型恐惧**: 怕走夜路、感觉被盯着
**类型**: 注视

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 被盯着、黑暗里有东西 |
| **符号化** | 眼睛 |
| **夸张** | 眼睛很大，在黑暗中发光 |
| **极简** | 只有眼睛，看不到其他 |

**最终形象**:
- 楼道黑暗角落，有**一双眼睛**在发光
- 眼睛跟着你移动
- 声控灯闪烁时眼睛更明显
- **看不到眼睛的主人**

**触发感受**:
- 晚上走楼道的恐惧
- 被盯着的感觉
- "黑暗里有东西"

**应对**:
- 快速通过
- 保持有光的区域
- 不要回头看

---

### 恐惧-003 围观的眼睛们

**原型恐惧**: 被孤立、被围观
**类型**: 包围

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 被看、被围观、被排斥 |
| **符号化** | 很多眼睛 |
| **夸张** | 眼睛从四面八方围过来 |
| **极简** | 只有眼睛，没有身体 |

**最终形象**:
- 多双**眼睛**从黑暗中浮现
- 它们在看你，围成圈
- 包围圈慢慢收缩
- **每双眼睛都是独立的，没有脸/身体**

**触发感受**:
- 被孤立的经历
- "他们在看我"
- 不合群的恐惧

**应对**:
- 不要被包围
- 找到缝隙逃脱
- 躲起来

## 3.3 学校场景 - 恐惧设计

### 恐惧-004 红叉

**原型恐惧**: 考试焦虑、成绩不好
**类型**: 包围/追赶

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 错误、不及格、分数 |
| **符号化** | 红色的叉 |
| **夸张** | 叉变成巨大的符号，在空中飞 |
| **极简** | 只有红叉，试卷质感作为身体 |

**最终形象**:
- **巨大的红色叉号**在空中飘
- 叉号有"试卷"的质感
- 越来越多，形成漩涡
- **没有"人"的形态，只是符号**

**触发感受**:
- 考试前的紧张
- 成绩不好的恐惧
- 被比较的压力

**应对**:
- 从红叉的缝隙穿过
- 躲到课桌下
- 不必"消灭"

### 恐惧-005 千眼

**原型恐惧**: 被发现做错事
**类型**: 注视

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 被监视、被发现 |
| **符号化** | 无数的眼睛 |
| **夸张** | 全身长满眼睛 |
| **极简** | 黑色剪影+眼睛，没有其他细节 |

**最终形象**:
- **黑色的人形剪影**
- **全身长满了眼睛**——从头到脚
- 所有眼睛都在看你
- 慢慢向你移动

**触发感受**:
- 做错事怕被发现
- 被监视的感觉
- "有无数双眼睛在看我"

**应对**:
- 必须绕行
- 利用环境遮挡
- 躲起来等它移开

### 恐惧-006 看打者（精英恐惧）

**原型恐惧**: 老师体罚、权威恐惧
**类型**: 注视+追赶

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 看（发现你做错事）+ 打（戒尺惩罚） |
| **符号化** | 巨大的眼睛 + 戒尺变成的武器 |
| **夸张** | 眼睛代替头部，戒尺变成大砍刀 |
| **极简** | 黑色西装剪影，没有其他细节 |

**最终形象**:
- **头部**: 一个**巨大的眼睛**，代替正常头颅
- **身体**: 黑色西装剪影，没有细节
- **右手**: 一把**戒尺形状的大砍刀**
- 巡视时眼睛四处转动
- 发现你后举着戒尺追赶

**触发感受**:
- 被老师批评的经历
- 怕被发现做错事
- 体罚的恐惧

**应对**:
- 必须躲避
- 利用环境遮挡
- 躲进柜子/桌子下

---

### 恐惧-007 指责之手（精英恐惧）

**原型恐惧**: 家长生气、被指责
**类型**: 注视+追赶

**设计方法论应用**:

| 分析 | 内容 |
|------|------|
| **核心概念** | 指责（食指指着你）+ 骂（张开的大嘴） |
| **符号化** | 巨大的食指 + 张开的大嘴 |
| **夸张** | 手和嘴比身体还大 |
| **极简** | 其他部位黑色剪影 |

**最终形象**:
- **头部**: 一张**张开的大嘴**，不断发出模糊的骂声
- **右手**: 一根**巨大的食指**，永远指着你
- **身体**: 黑色剪影，很小
- 一边骂一边追

**触发感受**:
- 被骂的经历
- "你做错了"
- 羞耻感

**应对**:
- 跑开
- 躲起来
- 不能被指到

# 四、"噩梦"设计（替代BOSS概念）

## 4.1 噩梦的本质

**噩梦 = 童年最深的恐惧，以最夸张的符号组合出现。**

噩梦不是单个符号，而是**多个恐惧符号的组合体**。

---

## 4.2 童年小巷 - 噩梦：巨手

### 原型恐惧
童年最怕的"那个大人"——可能是生气的家长、可怕的陌生人、某个创伤记忆中的人。

### 设计方法论应用

| 分析 | 内容 |
|------|------|
| **核心概念** | 抓住、惩罚、无法逃脱 |
| **符号化** | 无数巨大的手 + 眼睛 + 嘴 |
| **夸张** | 巨大到填满视野 |
| **极简** | 黑暗为背景，符号为主体 |

### 最终形象

**阶段变化**:

| 阶段 | 形象 | 描述 |
|------|------|------|
| 出现 | 一只巨手 | 从黑暗中伸出一只比房子还大的手 |
| 追逐 | 手+眼睛 | 手心睁开一只巨大的眼睛，盯着你 |
| 高潮 | 多手+多眼+嘴 | 无数手从四面八方伸出，中间是一张骂人的大嘴 |

**视觉细节**:
- 主色调：黑色背景 + 皮肤色手 + 黄色眼睛 + 红色嘴
- 手指在抓挠、收缩
- 眼睛死死盯着你
- 嘴在骂（听不清内容）

### 交互机制

| 阶段 | 现象 | 玩家行动 |
|------|------|----------|
| 阶段1 | 巨手出现 | 躲藏、逃跑 |
| 阶段2 | 手+眼睛追逐 | 在场景中寻找躲避点 |
| 阶段3 | 多手包围 | 熬过一段时间 |
| 阶段4 | 恐惧消退 | 手缩小、变淡 |

### 结局
- 巨手开始**缩小、变淡**
- 最终变成一只普通大小的手（某个温暖的手）
- 暗示：**原来没那么可怕**

---

## 4.3 学校 - 噩梦：审判者

### 原型恐惧
考试、被评判、被比较——学校里最大的恐惧。

### 设计方法论应用

| 分析 | 内容 |
|------|------|
| **核心概念** | 分数、叉号、名字、评判 |
| **符号化** | 红叉 + 红圈 + 名字 + 眼睛 |
| **夸张** | 填满整个教室 |
| **极简** | 没有人形，只有符号 |

### 最终形象

**形象构成**:
- **头部**: 一个**巨大的红叉**
- **身体**: 由**无数试卷**组成的飘动物体
- **眼睛**: 试卷上浮现的眼睛
- **声音**: 广播里的声音在念名字和分数

**视觉细节**:
- 主色调：白色试卷 + 红色叉号/圈
- 试卷在空中飘动形成漩涡
- 每张试卷上都有你的名字

### 交互机制

| 阶段 | 现象 | 玩家行动 |
|------|------|----------|
| 阶段1 | 试卷风暴形成 | 躲避、寻找掩体 |
| 阶段2 | 红叉追击 | 穿过缝隙 |
| 阶段3 | "选择题"出现 | 打破规则 |

### 核心机制
- 试卷上有"选择题"——选择"错误"的答案才是正确的
- 不是战斗，是**打破被定义的框架**

### 结局
- 风暴消散
- 红叉变成普通的勾
- 暗示：**分数没那么重要**

---

## 4.4 家庭 - 噩梦：缺失的相框

### 原型恐惧
家庭——某种缺失、某种不安、说不出的恐惧。

### 设计方法论应用

| 分析 | 内容 |
|------|------|
| **核心概念** | 缺失、空白、想不起来 |
| **符号化** | 空白的相框 + 吸力 |
| **夸张** | 相框越来越大 |
| **极简** | 只有相框，其他都被"吸"进去 |

### 最终形象

**形象构成**:
- 一个**巨大的空白相框**
- 相框在"吸"东西——周围的物品在飘向它
- 相框里是**一片空白**——你看不到原本应该在那里的人

### 交互机制
- 相框会"吸走"你的记忆碎片
- 需要主动放入正确的碎片
- 不是战斗，是"拼图"

### 结局
- 相框中出现一张**模糊的照片**
- 但依然看不清是谁的脸
- 暗示：**记起来了一些，但还不够**

---

## 4.5 最终 - 噩梦：镜中的自己

### 原型恐惧
自己——忘记了自己是谁。

### 设计方法论应用

| 分析 | 内容 |
|------|------|
| **核心概念** | 自己、记忆、遗忘 |
| **符号化** | 镜子 + 模糊的自己 |
| **夸张** | 镜子占据整个空间 |
| **极简** | 只有镜子和你 |

### 最终形象

**形象构成**:
- 一面**巨大的镜子**
- 镜子里是**模糊的你**——但看不清脸
- 周围漂浮着**记忆碎片**

### 交互机制

**这是非战斗的最终关卡：**

| 阶段 | 内容 | 玩家行动 |
|------|------|----------|
| 对视 | 看着镜中的自己 | 观看漂浮的记忆碎片 |
| 提问 | "你还记得什么？" | 选择回答（或选择不回答） |
| 接纳 | "忘记了也没关系" | 最终选择 |

### 结局分支

1. **接受遗忘**: 记忆碎片飘散，归于平静
2. **努力回忆**: 部分记忆恢复，但仍是模糊的
3. **选择离开**: 放弃寻找，回到开始

### 核心理念
最终噩梦不是要被打败的敌人，而是要被**接受**的自己。

---

## 4.6 噩梦设计总结

| 关卡 | 噩梦 | 核心概念 | 符号组成 | 交互本质 |
|------|------|----------|----------|----------|
| 童年小巷 | 巨手 | 抓住/惩罚 | 多手+眼睛+嘴 | 躲避/熬过去 |
| 学校 | 审判者 | 评判/分数 | 红叉+试卷+眼睛 | 打破规则 |
| 家庭 | 缺失的相框 | 缺失/遗忘 | 空白相框 | 拼图/填充 |
| 最终 | 镜中的自己 | 自己/遗忘 | 镜子+模糊身影 | 选择/接受 |

# 五、角色系统总结

## 5.1 设计方法论

```
┌─────────────────────────────────────────────────────────────┐
│                    恐惧抽象设计方法论                        │
├─────────────────────────────────────────────────────────────┤
│  步骤1: 提取核心概念                                         │
│  - 这个恐惧对象最可怕的1-2个概念是什么？                      │
│  - 例如：老师 → "看" + "打"                                  │
├─────────────────────────────────────────────────────────────┤
│  步骤2: 视觉符号化                                           │
│  - 把概念转化为视觉符号（眼睛、手、嘴、叉号等）               │
│  - 例如："看" → 眼睛，"打" → 戒尺                            │
├─────────────────────────────────────────────────────────────┤
│  步骤3: 夸张放大                                             │
│  - 符号部位不成比例地巨大                                     │
│  - 例如：眼睛变成头的大小，戒尺变成大砍刀                      │
├─────────────────────────────────────────────────────────────┤
│  步骤4: 极简其他                                             │
│  - 非核心部位用最简单的方式处理（黑色剪影）                   │
│  - 例如：身体只是黑色西装剪影，没有细节                       │
└─────────────────────────────────────────────────────────────┘
```

## 5.2 角色层级

```
┌─────────────────────────────────────────┐
│        主角 - "你"                       │
│        (无面孩童剪影)                    │
├─────────────────────────────────────────┤
│  存在层 - 记忆中的温暖                   │
│  (摇椅、手、影子、背影、光)              │
│  【没有人形，只有符号】                  │
├─────────────────────────────────────────┤
│  恐惧层 - 恐惧的符号化                   │
│  (手、眼睛、红叉、巨眼头+戒尺刀)         │
│  【符号夸张，身体剪影】                  │
├─────────────────────────────────────────┤
│  噩梦层 - 多符号组合                     │
│  (多手+眼睛+嘴 / 红叉+试卷 / 镜子)       │
│  【最夸张的符号组合】                    │
└─────────────────────────────────────────┘
```

## 5.3 设计要点

| 传统游戏 | 本游戏 | 差异 |
|----------|--------|------|
| NPC = 具体的人 | 存在 = 符号组合 | 没有人形，只有概念符号 |
| 敌人 = 怪物 | 恐惧 = 恐惧符号的夸张 | 概念→符号→夸张 |
| BOSS = 强敌 | 噩梦 = 符号组合体 | 多个符号的组合 |
| 击杀 = 胜利 | 熬过/理解/接受 | 不是消灭 |

## 5.4 核心原则

1. **没有人形** - 所有角色都是符号的组合，不是完整的人
2. **概念驱动** - 每个设计都从"恐惧概念"出发
3. **夸张对比** - 核心符号巨大，其他极简
4. **情感共鸣** - 符号对应童年真实的恐惧体验
