## 使用说明

1. 复制每个 Prompt 到 Midjourney
2. 生成后下载 PNG 格式
3. 使用图片编辑软件去除背景（如 Photoshop、Photopea）
4. 调整尺寸至推荐规格
5. 导入 RPG Maker MZ
