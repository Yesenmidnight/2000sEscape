# Midjourney 角色生成清单 - 逃离千禧年 (2000sEscape)

**版本**: 1.1
**日期**: 2026-02-08
**用途**: 主角、恐惧、噩梦角色的Midjourney生成prompt
**设计方法论**: 恐惧概念提取 → 视觉符号化 → 夸张放大 → 极简其他

---

## 通用风格后缀

所有角色prompt建议添加以下后缀以保持风格统一：

```
weirdcore dreamcore atmosphere, low saturation colors, vhs static filter, retro CRT aesthetic, pixel art style, 2D game character sprite, isolated on solid white background, subject only, no background elements, clean cutout --no background
```
