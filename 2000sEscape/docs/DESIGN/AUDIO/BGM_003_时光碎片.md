# BGM-003: 时光碎片

## 基本信息

| 属性 | 值 |
|------|-----|
| 编号 | BGM-003 |
| 名称 | 时光碎片 |
| 用途 | 关卡选择/回忆节点选择界面 |
| 目标时长 | 60-90秒 |
| 循环 | 是 |
| 优先级 | P1 (次要) |

## 使用场景

**关卡选择界面**
- 玩家从梦境大厅选择要进入的回忆节点
- 展示可用的关卡、难度、已收集物品等信息
- 停留时间相对较短

**场景描述**:
> 漂浮的记忆碎片在眼前排列，每个碎片都是一个被封存的时刻。触碰它们，会泛起涟漪。有些碎片明亮温暖，有些则暗淡模糊。你在选择，要回到哪一段记忆中去。

## 风格参考

### 参考风格
- **Minimal Ambient**: 极简的氛围
- **Music Box**: 音乐盒的怀旧感
- **Cinematic Ambient**: 简短的电影感

### 参考作品
- *Hollow Knight* 的地图界面音乐
- *Celeste* 的章节选择
- *Night in the Woods* 的菜单切换

### 配器设想
```
核心:
- 音乐盒音色 (主要旋律)
- 柔和的合成器 Pad
- 水滴/涟漪的音效

氛围:
- 轻微的混响
- 玻璃质感的音色
- 偶尔的风铃声

点缀:
- 极简的钢琴音符
- 柔和的弦乐长音
```

## 循环设计

```
[前奏 4s] → [A段 16s] → [B段 16s] → [A'段 16s] → 循环
```

**设计要点**:
- 因为停留时间短，循环周期可以较短
- 需要有"选择"的感觉，但不要太紧张
- 可以加入UI交互的音效设计考虑

## 与UI音效的配合

此界面会有较多UI交互，音乐需要为音效留出空间:

```
音乐频率规划:
- 低频: 环境铺垫 (不过多)
- 中频: 主要旋律 (音乐盒)
- 高频: 留给UI音效使用
```

**建议音效**:
- 选项悬停: 柔和的风铃声
- 选项确认: 轻微的涟漪声
- 切换关卡: 翻页/水波声
