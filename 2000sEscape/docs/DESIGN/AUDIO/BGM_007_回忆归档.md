# BGM-007: 回忆归档

## 基本信息

| 属性 | 值 |
|------|-----|
| 编号 | BGM-007 |
| 名称 | 回忆归档 |
| 用途 | 关卡结算界面 |
| 目标时长 | 60-90秒 |
| 循环 | 是 (但可以是完整歌曲形式) |
| 优先级 | P1 (次要) |

## 使用场景

**结算界面**
- 玩家成功撤离后的结算
- 展示搜集的物品、回忆价值、统计数据
- 解锁成就提示
- 带有怀旧物品描述

**场景描述**:
> 你从记忆中带回了这些碎片。泛黄的照片、旧玩具、褪色的红包...每一样都带着一段故事。它们被轻柔地放入档案，成为你走出梦境的又一步。屏幕上浮现温暖的文字，这是属于你的回忆。

## 风格参考

### 参考风格
- **Lo-fi Chill**: 放松的低保真
- **Nostalgic Ambient**: 怀旧氛围
- **Comfort Music**: 舒适音乐

### 参考作品
- *Stardew Valley* 的结算/收获
- *Animal Crossing* 的物品收集
- *Firewatch* 的温情时刻
- *Gris* 的治愈感

### 配器设想
```
核心:
- 温暖的钢琴 (主要)
- 柔和的吉他 (可选)
- Lo-fi 鼓点 (轻柔)

氛围:
- 温暖的合成器 Pad
- 磁带噪音
- 轻微的黑胶质感

点缀:
- 音乐盒
- 柔和的弦乐
- 环境采样 (温暖的室内感)
```

## 结算流程与音乐配合

```
撤离成功
    │
    ↓ 淡入结算音乐
    │
┌───────────────────────┐
│  物品展示              │ ← 音乐渐入，温馨
│  (逐个展示搜集物品)     │
└───────────┬───────────┘
            │
            ↓
┌───────────────────────┐
│  价值汇总              │ ← 音乐保持
│  (数字累积动画)        │
└───────────┬───────────┘
            │
            ↓
┌───────────────────────┐
│  统计数据              │ ← 音乐保持
│  (时间、搜集率等)      │
└───────────┬───────────┘
            │
            ↓
┌───────────────────────┐
│  解锁提示              │ ← 音乐可以有小的庆祝感
│  (如有新解锁)          │
└───────────┬───────────┘
            │
            ↓ 返回梦境大厅
```

## 循环设计

```
[温馨前奏 6s] → [A段 16s] → [B段 16s] → [A'段 16s] → [C段 16s] → 循环/结束
```

**设计要点**:
- 因为结算时间相对固定，可以是完整歌曲
- 也可以设计成循环，供长时间浏览
- 需要有"归档"的仪式感

## SUNO Prompt

### 主Prompt (推荐)

```
Style: Lo-fi chill, nostalgic, warm piano, comfort music, sentimental
Instruments: Soft piano, gentle acoustic guitar, lo-fi hip hop beats, warm synth pads, vinyl crackle
Mood: Warm achievement, bittersweet nostalgia, peaceful satisfaction, cozy memories
Tempo: 70-80 BPM, relaxed and content
Structure: Gentle build, warm resolution, looping friendly
Production: Lo-fi warmth, tape saturation, soft compression, intimate room sound
Description: Settlement and reward screen music for a nostalgic game. You've successfully escaped with memories. Warm piano melody with gentle lo-fi beats. The feeling of looking through a box of old photos - bittersweet but mostly warm. Should make players feel accomplishment and nostalgia together.
```

## 与UI的配合

```
视觉元素              音乐氛围
────────────────────────────────────
物品逐个出现    ←→    音乐平稳进行
数字累积        ←→    节奏轻快但不急

```
