# 室内元件需求文档 - 逃离千禧年 (2000sEscape)

**版本**: 1.0
**更新日期**: 2026-02-08
**用途**: 可进入建筑的室内场景美术需求

---

## 概述

本文档涵盖所有**可进入建筑**的室内场景元件，是对户外元件文档的补充。

### 设计原则

1. **生活气息** - 每个房间都应该有"有人生活过"的痕迹
2. **时代特征** - 2000年代初中国家庭/场所的真实细节
3. **怪核氛围** - 熟悉却空旷，温馨中透着诡异
4. **可交互性** - 大多数家具可翻找、可互动

## 门/门框

### 元件信息
- **类型**: 通用组件
- **用途**: 房间入口、过渡点

### 视觉描述
老式木门或铁门，门框有贴春联的痕迹。门把手是圆形或长条形金属，可能有些松动。门上有气窗（上方的小窗户）。木门漆面斑驳，铁门有锈迹。

### 变体
| 变体 | 描述 |
|------|------|
| 木门 | 棕色/深红色，常见于室内房间 |
| 铁防盗门 | 绿色/蓝色，常见于单元门 |
| 纱门 | 夏天用的纱网门 |
| 折叠门 | 塑料或木制折叠门 |

### Midjourney Prompt


### 游戏用途
- 房间进入/退出的交互点
- 门后可能有隐藏物品

## 墙面/墙纸

### 元件信息
- **类型**: 通用背景
- **用途**: 室内背景基底

### 视觉描述
2000年代中国家庭常见的墙面：白色涂料、浅色墙纸（花卉/几何图案）、或半墙瓷砖（厨房浴室）。墙上有开关插座、挂画痕迹、钉子孔。可能有些裂缝或水渍。

### 变体

| 几何墙纸 | 条纹/格子图案 | 客厅 |
| 半墙瓷砖 | 下半部分贴瓷砖 | 厨房、浴室 |
