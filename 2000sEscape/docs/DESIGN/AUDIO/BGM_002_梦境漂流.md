## 使用场景

**梦境大厅 (Dream Lobby)**
- 玩家在关卡之间的休憩空间
- 功能: 记忆等级查看、武器商店、技能树、回忆展览
- 玩家可能停留较长时间，需要特别耐听

**场景描述**:
> 一个永恒黄昏的空间，漂浮着零散的记忆碎片。老式电视机堆叠在角落，播放着没有信号的雪花。远处有一扇扇发光的门，通向不同的回忆。这里是安全的，但有一种"你不属于这里"的微妙感。

**核心情绪**: "安全的孤独"——这里没有人，但你并不害怕，只是感到一种永恒的安静

## 风格参考

### 参考风格
- **Space Ambient**: 太空漂流的空旷感
- **Drone Music**: 持续的长音，时间停滞感
- **Dream Ambient**: 梦境特有的朦胧感

### 参考作品
- *Yume Nikki* 的BGM风格
- *Bioshock* 安全区域的氛围
- *Silent Hill* 的安静探索音乐
- *Animal Crossing* 博物馆的平静感（但更孤独）

### 配器设想
```
核心:
- 合成器 Pad (宽广、空灵)
- 低频 Drone (持续的基础音)
- 偶尔的钢琴音符 (极简、随机)

氛围:
- 大量混响 (空间感)
- 磁带循环 (Tape Loop)
- 柔和的白噪音
- 远处的钟声

纹理:
- 柔和的合成器和弦
- 非常缓慢的节奏 (或无节奏)
- 偶尔的环境音 (风声、电流声)
```

## 循环设计

```
[环境前奏 12s] → [A段 32s] → [B段 32s] → [A'段 32s] → [C段 32s] → [D段 32s] → 循环
```

**设计要点**:
- 因为停留时间长，需要更长的循环周期避免重复疲劳
- 可以是"非确定性"的，即每次循环略有不同
- 整体保持平稳，不要有明显的高潮

## 动态分层设计

梦境大厅可以支持根据玩家状态调整音乐:

```
基础层 (始终):
- Drone + Pad
- 环境噪音

交互层 (打开菜单时):
- 加入轻微的音效响应
- 钢琴音符变得更频繁

特殊层 (首次进入/重要事件):
- 加入旋律性元素
- 更丰富的和声
```

## 验收标准

### 必须满足
- [ ] 时长 ≥ 180秒，可循环
- [ ] 不产生重复疲劳感

- [ ] 整体平稳，无突兀变化

- [ ] 有明显的空间感/漂浮感

- [ ] 有记忆/怀旧元素
- [ ] 适合分层混音
