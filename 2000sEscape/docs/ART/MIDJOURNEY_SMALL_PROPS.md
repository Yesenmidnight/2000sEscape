# Midjourney 小型道具元件 - 可拾取物品

**版本**: 1.0
**日期**: 2026-02-08
**用途**: 可拾取、可放入背包的小型道具（占用1-4格）
**尺寸规格**: 1×1, 1×2, 2×1, 2×2, 1×3

---

## 生成说明

小型道具元件特点：
- **透明背景** - 使用 `isolated on solid white background, subject only, no background elements --no background`
- **俯视图/侧视图** - 根据用途选择视角
- **可拾取** - 看起来是独立物体
- **回忆价值** - 每个物品都有童年记忆关联
