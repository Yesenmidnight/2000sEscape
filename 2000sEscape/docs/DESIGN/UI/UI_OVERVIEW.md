# UI设计文档 - 探索界面

> 横版2D搜打撤游戏的探索界面设计

## 1. 玩家状态面板 (左下角)

### 显示内容
| 属性 | 描述 | 机制 |
|------|------|------|
| **HP 血量** | 玩家生命值 | 归零 = 玩家死亡 |
| **ST 精力** | 体力值 | 控制奔跑和跳跃，缓慢恢复 |

### 精力系统
```
精力消耗:
- 奔跑: 每秒消耗 X 点
- 跳跃: 每次消耗 Y 点

精力恢复:
- 站立/行走时: 每秒恢复 Z 点
- 奔跑/跳跃时: 不恢复
```

---

## 2. 主动技能栏 (中间)

### 基本信息
- **格子数量**: 3格
- **技能来源**: 装备附带
- **切换方式**: 装备变化自动更新

### 技能类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **主动技能** | 需要按键触发 | 抓拍照片、冲刺、格挡 |
| **被动技能** | 自动生效 | 增加移速、减少伤害 |
| **纯数值** | 无技能，仅属性加成 | +10攻击力 |

### 技能示例: 运动相机
```
装备: 运动相机
技能: 抓拍照片
效果: 闪光灯让面朝方向所有敌人静止1秒
冷却: 10秒
```

---

## 3. 快捷道具栏 (右侧)

### 基本信息
- **格子数量**: 4格快捷栏 + 1格背包按钮
- **快捷键**: 1, 2, 3, 4
- **背包按钮**: 打开完整背包界面

### 操作方式
```
按键1 → 使用快捷栏第1格道具
按键2 → 使用快捷栏第2格道具
按键3 → 使用快捷栏第3格道具
按键4 → 使用快捷栏第4格道具
点击📦 → 打开背包界面
```

## 4. 搜索系统

### 容器搜索
```
玩家靠近容器 → 按交互键 → 弹出搜索窗口
┌─────────────────────┐
│   📦 旧木箱         │
├─────────────────────┤
│ [旧玩具车]  x1      │
│ [褪色照片]  x1      │
├─────────────────────┤
│   [全部拾取]        │
└─────────────────────┘
```

### 直接拾取
```
玩家触碰地面物品 → 自动捡起 → 尝试放入背包
成功 → 物品消失 + 提示音
失败 → 提示"背包已满"
```

## 5. 背包系统

### 设计参考
- 《背包乱斗》(Backpack Hero)
- 《逃离塔科夫》(Escape from Tarkov)

### 核心机制

#### 不规则道具摆放
```
每个道具有不同形状和占格:

小物品 (1格):     中物品 (2格):     大物品 (4格):
┌───┐            ┌───┬───┐        ┌───┬───┐
│ 💊│            │ 🔫│   │        │ 📦│   │
└───┘            └───┴───┘        ├───┼───┤
                                    │   │   │
                                    └───┴───┘
```

#### 背包格子示例
```
┌───┬───┬───┬───┬───┬───┐
│ 💊│ 🔫│   │   │ 📦│   │
├───┼───┤   │   ├───┼───┤
│   │   │   │   │   │   │
├───┼───┼───┼───┼───┼───┤
│   │   │   │   │   │   │
├───┼───┼───┼───┼───┼───┤
│   │   │   │   │   │   │
└───┴───┴───┴───┴───┴───┘
```

#### 自动摆放逻辑
```
玩家拾取物品时:
1. 检测物品占格形状
2. 扫描背包寻找最大连续空格区域
3. 尝试自动摆放
   - 成功 → 物品放入背包
   - 失败 → 提示"背包已满"
```

## 6. 战斗系统

### 战斗风格
- 参考《魂斗罗》
- 2D动作动画效果

### 基本操作
```
攻击键 → 面朝方向攻击
方向+攻击 → 指定方向攻击
跳跃+攻击 → 空中攻击
```

| 优先级 | 功能 | 复杂度 |
|--------|------|--------|
| P0 | 玩家状态显示 | 低 |
| P0 | 快捷道具栏基础 | 中 |
| P1 | 主动技能栏 | 中 |
| P1 | 容器搜索窗口 | 中 |
| P1 | 直接拾取物品 | 低 |
| P2 | 不规则背包系统 | 高 |
| P2 | 自动摆放检测 | 高 |
| P2 | 战斗动画效果 | 高 |
