# BGM-001: 记忆入口

## 基本信息

| 属性 | 值 |
|------|-----|
| 编号 | BGM-001 |
| 名称 | 记忆入口 |
| 用途 | 主菜单背景音乐 |
| 目标时长 | 90-120秒 |
| 循环 | 是 |
| 优先级 | P0 (核心) |

## 使用场景

**主菜单界面**
- 游戏启动后的第一个界面
- 玩家看到游戏标题、开始游戏、继续游戏、设置等选项
- 可能停留时间较长，需要耐听

**场景描述**:
> 屏幕渐亮，一个模糊的千禧年街景作为背景。老式电视机的雪花噪点轻轻跳动。远处传来模糊的童声笑语，像是隔着很厚的玻璃。一种"要回家了，但又不知道家在哪里"的感觉。

## 风格参考

### 参考风格
- **Lo-fi Hip Hop**: 柔和的节拍、低保真质感
- **Dark Ambient**: 低沉的氛围铺垫
- **Vaporwave**: 千禧年的怀旧电子感

### 参考作品
- *Lofi Girl* 系列的安静感
- *Disco Elysium* 主菜单的沉浸感
- *Omori* 的梦境开场氛围
- *Milk inside a bag of milk* 的低保真舒适感

### 配器设想
```
核心:
- 柔和的电钢琴 (Rhodes风格)
- 远处的人声采样 (模糊的童声)
- Lo-fi 鼓点 (非常轻，几乎只是点缀)

氛围:
- 磁带噪音 (Tape Hiss)
- 远处的电视声
- 柔和的合成器 Pad
- 偶尔的音乐盒音色

可选:
- 轻微的雨声
- 时钟滴答声
```

## 循环设计

```
[前奏 8s] → [A段 16s] → [B段 16s] → [A'段 16s] → [C段 16s] → 循环回到A段
```

**设计要点**:
- 前奏需要有"渐入"的感觉，配合游戏启动
- 循环点要自然，不能突兀
- 避免明显的旋律高潮，保持平缓

## SUNO Prompt

### 主Prompt (推荐)

```
Style: Lo-fi ambient, dreamy, nostalgic, Chinese 2000s childhood memory, vaporwave aesthetic
Instruments: Soft electric piano, tape hiss, distant children voices sample, gentle synth pads, music box, minimal lo-fi beats
Mood: Melancholic nostalgia, bittersweet, dreamy, slightly eerie comfort, isolated but warm
Tempo: 60-70 BPM, very slow and relaxed
Structure: Atmospheric intro, gentle loop with subtle variations
Production: Lo-fi, tape saturation, VHS aesthetics, vinyl crackle, reverb
Description: Main menu theme for a pixel art game about being trapped in year 2000 memories. The feeling of looking at old photos - familiar yet distant. Soft Rhodes piano melody with ambient textures. Like hearing a lullaby through walls.
```

## 验收标准

### 必须满足
- [ ] 时长 ≥ 90秒，可循环
- [ ] 整体音量平稳，无明显突兀
- [ ] 怀旧感明确传达
- [ ] 循环点自然流畅
- [ ] 不会过于悲伤或过于快乐

### 优先满足
- [ ] 有明显的"千禧年"年代感
- [ ] 适合长时间播放不疲劳
- [ ] 有轻微的不安/神秘感

### 加分项
- [ ] 有中国童年记忆的元素
- [ ] 有VHS/磁带质感的细节
- [ ] 情绪层次丰富
