# BGM-006: 黄昏小巷 - 撤离

## 基本信息

| 属性 | 值 |
|------|-----|
| 编号 | BGM-006 |
| 名称 | 黄昏小巷 - 撤离 |
| 用途 | 童年小巷关卡 - 撤离点倒计时阶段 |
| 目标时长 | 60-90秒 |
| 循环 | 是 (或渐进至高潮) |
| 优先级 | P0 (核心) |

## 使用场景

**童年小巷 - 撤离阶段**
- 玩家到达撤离点，开始"加载回忆"倒计时
- 敌人开始向撤离点聚集
- 时间有限，必须在倒计时结束前坚持住

**场景描述**:
> 找到了出口。一扇发光的门矗立在巷子尽头。你冲了过去，但身后的黑影也察觉了你的意图。门需要时间打开，你必须在这几秒钟里坚持住。越来越近了...再坚持一下...

## 情绪定位


**核心情绪**: "最后的冲刺"——困难但充满希望，再坚持一下就能逃出去

## 风格参考

### 参考风格
- **Glitch Vaporwave**: 故障蒸汽波 - 损坏但美丽的逃离感
- **Broken Hope**: 破碎的希望 - 在故障中看到光芒
- **Y2K Escape**: 千禧年逃逸美学 - 早期数字时代的"加载中"感

### 参考作品
- *Resident Evil* 的撤离倒计时 (但更lo-fi)
- *Silent Hill* 的逃脱场景
- *Petscop* 的诡异希望感
- 早期游戏"Loading"界面的等待音乐
- Vaporwave中带"希望"元素的作品

### 配器设想
```
核心:
- 故障的驱动节奏 (但比警报更规律，像倒计时)
- 渐进式"修复"的旋律 (从失真慢慢变清晰)
- "希望"的音色 - 像损坏的文件终于要加载完成

氛围:
- 磁带从损坏慢慢变正常的感觉
- 加载进度条的听觉化
- 远处"光"的声音 - 带着VHS的光晕感
- 断续但越来越连贯的节奏

动态:
- 开始时更多故障/卡顿
- 随时间推移故障减少
- 最后阶段旋律最清晰
- 但始终保持一点点"损坏"的质感
```

## 渐进式设计

撤离阶段有时间倒计时，音乐可以随时间变化:

```
时间轴设计 (假设30秒撤离):

0-10秒: 基础紧张
- 节奏开始
- 基本的紧张氛围
- 撤离点音效

10-20秒: 加强
- 加入更多节奏层
- 音量略微提升
- 旋律开始出现

20-30秒: 冲刺
- 全部乐器加入
- 紧张度达到顶峰
- "希望"的旋律片段

成功: 快速解决音效
失败: 截断至警报音乐
```

## SUNO Prompt

### 主Prompt (推荐 - 故障蒸汽波)

```
Style: Glitch vaporwave escape, broken hope, Y2K extraction, corrupted countdown, lo-fi survival
Instruments: Stuttering but hopeful synth pads, tape degradation texture, broken-but-mending melody, VHS ambience, gradual clarity emerging from static
Mood: Urgent hope through static, file finally loading, corrupted memory being restored, almost there despite glitches
Tempo: 120-130 BPM with gradual stabilization feel
Structure: Progressive build from glitchy to clearer, countdown tension with hopeful undertones
Production: Lo-fi tape warmth, sample rate that seems to improve, VHS tracking stabilizing, bitcrush that lessens over time
Description: Extraction countdown music with vaporwave aesthetics. You've reached the exit but it needs to "load." The music starts more broken and glitchy, gradually becoming clearer and more hopeful as the exit opens. Like a corrupted save file finally loading. Urgent pulse but with warmth. The feeling of "just a few more seconds" through the lens of year 2000 digital nostalgia. Not clean and modern - warm, damaged, but ultimately hopeful.
```

```
Style: Loading screen hope, buffer progression, digital dawn, corrupted file recovery
Instruments: Progress bar synth pulse, dial-up melody fragments (but pleasant), loading chimes, data recovery ambiance
Mood: Patient urgency, progress being made, anticipation, light at the end of the buffer
Tempo: 125 BPM, steady like a progress bar
Production: Early internet aesthetics, soft digital artifacts, warm lo-fi, gentle tape hiss
Description: Escape music that sounds like a nostalgic "loading" screen with hope. The urgency comes from wanting the progress to complete. Synth pulses that mark time passing. Melody fragments that become more complete as the song progresses. Like watching a download bar fill up in 2002 but your life depends on it completing.
```

### 备选Prompt 2 (更磁带修复感)

```

Instruments: Tape deck warming up, melody emerging from wow/flutter, progressing from warble to stable, warm pads
Mood: Something precious being recovered, urgency to save the memory, hope through patience
Tempo: 120 BPM, from unstable to stable
Production: Tape saturation, wow decreasing over time, flutter calming down, warmth increasing
Description: Escape music that sounds like a precious cassette tape being carefully played. Starts with heavy wow and flutter, gradually stabilizes into a clear, hopeful melody. The urgency of saving something before it's lost forever. Warm, nostalgic, with a ticking countdown feel. Like the tape player finally working after blowing into it.
```

## 与撤离点音效的配合

```
音乐元素              音效元素
────────────────────────────────────
节奏脉冲        ←→    倒计时滴答声
渐进加强        ←→    撤离点加载进度音
希望旋律        ←→    成功提示音
```

**建议音效**:
- 撤离点存在: 柔和的脉冲光声
- 倒计时: 低沉的滴答/心跳
- 进度: 充能般的上升音
- 成功: 明亮的解决音
